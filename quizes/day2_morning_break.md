# How would I slice _all_ even indexes?

- `[::2]`

- `[0::2]`

- `[1::2]`

- `[2::2]`

# What does this expression evaluate to?

```python
[1, 3, 5, 7, 9][1:-1]
```

- `[3, 5, 7]`

- `[1, 3, 5, 7]`

- `[2, 4, 6, 8]`

- `[1, 2, 3, 4, 5, 6, 7, 8]`
